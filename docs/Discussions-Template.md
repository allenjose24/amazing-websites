# Welcome to the Vault! Let's build together 🏛️✨

Hi everyone! 👋

Welcome to the **Amazing Websites (The Vault)** discussion board! This space is dedicated to curating premium design resources, interactive frontend animations, typography showcases, and developer tools. 

We want to make this repository a go-to directory for visual and creative developers—and **we need your help to grow and polish it!**

Whether you are here to browse, suggest your favorite web assets, report a bug, or write React components, this discussion is the starting point for collaboration.

---

## 🚀 How You Can Help

There are two primary ways you can get involved:

### 1. 📂 Suggest & Add Resources (No Code Required)
If you've found an amazing website, design framework, utility tool, or creative layout, suggest it to the index!
* **How to suggest**: Log into the application using your account, head to the navigation, click **Contribute**, and submit the form.
* **Review Process**: Submissions go to a queue where admins review and approve them before they are added to the live 3D card decks.
* **Want custom categories?** If you have an idea for a category that isn't listed (like *WebGPU*, *Three.js*, *SVG Filters*), comment below!

### 2. 💻 Frontend Page & Feature Development (Code Contributions)
The Vault's interface is built using a modern, high-performance stack: **React 19, Vite, Tailwind CSS, Framer Motion, and Lenis**. We welcome pull requests for:
* **Custom Category Layouts**: Designing new visual cards and interactive modes for new categories.
* **Micro-interactions**: Improving hover states, spring curves, or transition animations.
* **Mobile Responsiveness**: Enhancing layouts across tablet and mobile viewports.
* **Performance Optimizations**: Improving loading speeds, reducing CLS, or refining edge function runs.

To help you get started with the codebase, check out our **[Complete Technical Wiki Docs Hub](./docs/Home.md)**!

## 🛠️ Local Development Quickstart

If you want to contribute code:
1. Clone this repository.
2. Install dependencies: `npm install`
3. Configure your local environment variables in `.env.local`:
   ```env
   VITE_SUPABASE_URL=your-supabase-url
   VITE_SUPABASE_ANON_KEY=your-supabase-anon-key
   ```
4. Start the Vite local server: `npm run dev`
5. Open `http://localhost:5173` and start hacking!

We look forward to seeing your suggestions and contributions! Let’s build the ultimate web design repository together! 🚀🎨
