# Security Policy

## Supported Versions

We actively maintain the latest version of the application. Please ensure you are running the most up-to-date version of the repository.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please do **NOT** open a public issue. Doing so exposes the vulnerability to the public before it can be fixed.

Instead, please report it privately:
* **Contact Email**: **pottumkalallenjose@gmail.com**
* **Expected Response**: You will receive a response within 48 hours acknowledging your report, followed by updates on the fix.

Thank you for helping keep this project secure!
